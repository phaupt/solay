# Contributor Code of Conduct

This repository is for Solar Manager users, makers, and contributors who want to build or improve the dashboard. Participation in issues, pull requests, discussions, and other project spaces means agreeing to the following standards.

## Expected Behavior

- Be respectful, patient, and technically specific.
- Assume good intent and ask clarifying questions before escalating.
- Keep criticism focused on the code, design, or documentation, not the person.
- Share reproducible details when reporting problems.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks
- Discriminatory or sexualized language
- Trolling, deliberate disruption, or repeated bad-faith arguments
- Publishing someone else's private information without permission

## Maintainer Responsibilities

The maintainer may edit, remove, or reject comments, issues, pull requests, and other contributions that do not follow this code of conduct. Repeated or severe violations may lead to temporary or permanent bans from project spaces.

## Scope

This code of conduct applies in all repository spaces and when someone is representing the project in public.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer via GitHub at <https://github.com/phaupt>. Include links, screenshots, or other context that makes the report actionable.
